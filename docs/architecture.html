Factory Inventory Management — System Architecture

Vue 3 + FastAPI full-stack application · In-memory mock data · REST API

Tech Stack

Frontend · Port 3000

Vue 3 + Vite

Composition API · SPA
  • Vue Router 4 — client-side routing
  • Axios — HTTP client
  • Composables — shared state
  • i18n — English / Japanese
  • Scoped CSS — component styles
Backend · Port 8001

Python FastAPI

Uvicorn ASGI · Auto-reload
  • Pydantic v2 — data validation
  • CORS middleware — cross-origin
  • Auto docs — /docs, /redoc
  • uv — package manager
  • Python ≥ 3.11
Data Layer

In-Memory JSON

7 data files · No database
  • Loaded at startup via mock_data.py
  • Filtered in-memory per request
  • Changes don't persist
  • Demo / workshop use only
